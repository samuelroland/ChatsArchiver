File tree data
Details of the file tree for data of the server for ChatsArchiver


Here is an exemple of the structure with 2 users (John => username = john99 and Kate => username = kateP) and each have severals chats importated.

The chats have the name "chat-" + chatID.
All the other files have the name uniqid() + extension.

Every chat has a json called chat-2.json and a folder chat-2 on the side. Every chat is stored in the folder of the user.

the 3 folders at the root of model are data (contains all the data of the chats: chat content, images, audio, videos, other files, ... and an image for the profil of the user), tempsfiles (contains all the tempory files: chat in .txt not yet converted.) and users that just contains the list of the users.
